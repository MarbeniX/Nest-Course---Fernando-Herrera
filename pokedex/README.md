# Execute on developer mode

1. Clone the repository
2. Execute

```
npm install
```

3. You have to have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Rise the DB

```
docker-compose up -d
```

5. Reconstruct the databsae

```
http://localhost:3000/api/seed/
```
